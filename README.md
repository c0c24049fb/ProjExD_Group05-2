# カー走

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* どんどんスピードが上がるバイクで障害物を避けながらコインをゲットしろ！！！
![title](scr.png)
## ゲームの遊び方
* スペースキーか上矢印キーでジャンプ、右シフトを押すと20スコア使い一定時間無敵になれる！
* Rでゲームスタート、リスタート
## ゲームの実装
### 共通基本機能
* スペースで車が飛ぶ
* コインを拾うとスコアアップ
* スコアの表示
* 障害物

### 分担追加機能
* 無敵（担当：村田）:スコアを20使うと無敵状態になれる機能を実装
* サウンド、エフェクト（担当：星野）:BGMとコインをとった時の音、衝突音、ジャンプ音を追加した。
* スキン、背景画像制作（田中）背景とスキンを実装
* ホーム画面制作（きたむら）ホーム画面を実装
* 落とし穴制作（ひでしま）:落とし穴を実装
